# Batwa Travels - Landing (React + Vite)

Simple landing page for a travel & tours business built with React and Vite.

Getting started

1. Install dependencies

```powershell
cd Batwa_Travels-main
npm install
```

2. Run development server

```powershell
npm run dev
```

App structure

- `src/components` — UI components (`Banner`, `TravelCard`)
- `src/pages` — `Home` and `Tour` pages
- `src/data/tours.js` — sample tours data

What to do next

- Replace sample images/data with your own
- Hook the contact form to a backend or email service
